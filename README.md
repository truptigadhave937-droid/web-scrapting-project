Objective:
The objective of this project is to scrape product listing data from a publicly accessible website
and store it in a structured format using Python.

Website:
https://books.toscrape.com/

Description:
This project demonstrates web scraping using Python and BeautifulSoup.
The selected website is static and suitable for scraping without authentication.

Tools & Libraries Used:
- Python
- Requests
- BeautifulSoup
- Pandas

Data Extracted:
- Book Title
- Price
- Availability
- Rating

Approach:
1. Sent HTTP requests to fetch webpage content.
2. Parsed HTML using BeautifulSoup.
3. Identified product listing elements.
4. Extracted relevant book details.
5. Stored the data in a Pandas DataFrame.
6. Exported the data to an Excel file.

Output Files:
- books.xlsx
- books.csv (optional)

Result:
Successfully scraped book data and stored it in a structured Excel format.
This project fulfills the requirements of a basic web scraping assignment.
